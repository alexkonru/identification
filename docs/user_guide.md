# Руководство пользователя

## 1. Подготовка инфраструктуры
1. Создайте **зоны**.
2. Создайте **комнаты** внутри зон.
3. Добавьте устройства: камера, микрофон (опционально), замок.

## 2. Регистрация сотрудника
1. Откройте вкладку **Персонал и доступ**.
2. Нажмите **Добавить**.
3. Введите ФИО, снимите серию фото, запишите голос (если используется).
4. Сохраните пользователя.

## 3. Назначение прав
1. Выберите пользователя.
2. Отметьте доступные комнаты.
3. Нажмите **Сохранить права**.

## 4. Запуск системы
1. Откройте вкладку **Настройки**.
2. Нажмите **Запустить систему**.
3. Убедитесь, что состояние — **Работает**.

## 5. Мониторинг
1. Перейдите на вкладку **Мониторинг**.
2. Выберите нужную камеру.
3. Следите за кратким и системным логом.

## 6. Остановка
- Для остановки обработки нажмите **Остановить систему** во вкладке **Настройки**.
- Для остановки контейнеров используйте `./stop_docker.sh`.

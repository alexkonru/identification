# Справка по Biometry Admin

## Быстрый старт
1. Запустите сервисы: `./start_docker.sh`.
2. Откройте вкладку **Система** и проверьте статусы сервисов.
3. Во вкладке **Настройки** нажмите **Запустить систему**.
4. Перейдите в **Мониторинг** для просмотра точек прохода.

## Вкладки
- **Мониторинг**: поток с серверных камер и журнал pipeline-событий.
- **Система**: состояние gateway, БД, vision и audio.
- **Настройки**: запуск/остановка системы, тема и параметры интерфейса.
- **Персонал и доступ**: регистрация пользователей и права доступа.
- **Инфраструктура**: зоны, комнаты, устройства, сканирование железа.
- **Журнал**: история решений по доступу.

## Live-режим
- Идентификация работает только когда система запущена.
- Если система остановлена, мониторинг не обрабатывает кадры.

## Диагностика
- Логи gateway: `docker compose logs --tail=120 gateway-service`
- Логи vision: `docker compose logs --tail=120 vision-worker`
- Логи audio: `docker compose logs --tail=120 audio-worker`
